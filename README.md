Getting Started
---------------

Install gulp:

    npm install --global gulp-cli

Install dependencies:

    npm install
    
Run gulp:

    gulp watch
    
For firefox/opera/safari use flag:

    gulp watch --firefox
    
Go to the Extensions page and turn on Developer mode:

    chrome://extensions/
    
Load app folder

    Press 'Pack extension' button on page chrome://extensions/
    
Packing extension
